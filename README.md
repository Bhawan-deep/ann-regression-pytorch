# ANN Regression using PyTorch
## Overview
This project implements an Artificial Neural Network (ANN) for regression using PyTorch. The model predicts power plant energy output based on environmental features.
---
##Dataset Features
* AT (Temperature)
* V (Exhaust Vacuum)
* AP (Ambient Pressure)
* RH (Relative Humidity)
* PE (Energy Output - Target)
---
## Tech Stack
* Python
* PyTorch
* NumPy
* Pandas
* Matplotlib
* Scikit-learn
---
##  Model Performance
* R² Score: **~0.94**
* Test MSE: **~0.05**
